# car-dealership-website
Página de cadastro de veículos desenvolvida com HTML e CSS.  O projeto permite preencher um formulário com informações do carro, como preço, marca, modelo, quilometragem, descrição, opcionais e imagens do veículo.  Desenvolvido como parte dos meus estudos em desenvolvimento web para praticar a criação de formulários e estilização de páginas.
